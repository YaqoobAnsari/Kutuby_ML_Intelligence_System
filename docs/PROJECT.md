# Kutuby Dashboard Project

## Vision
Build an internal ML observability and data intelligence platform for Kutuby.

## Context
Two production pronunciation models:
1. Arabic letters (Wav2Vec-based).
2. Quranic words (Tarteel fine-tuned).

Every inference stores:
- Audio recording
- Metadata
- Prediction
- Score (Pass / Retry / Fail)

## Primary Users
- ML engineers
- Backend engineers
- Senior management

## Goals
- Observe model performance.
- Build a proprietary pronunciation dataset.
- Identify failure modes.
- Support future retraining.
